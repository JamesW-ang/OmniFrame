# 行为准则

## 我们的承诺

为了营造一个开放欢迎的社区环境，我们作为贡献者和维护者承诺，为项目和社区中的所有人提供一个无骚扰的体验，无论其年龄、身体条件、残障状况、种族、性别认同和表现、经验水平、国籍、个人外表、政治信仰、宗教信仰或性别认同和取向如何。

## 我们的标准

有助于创造积极环境的行为包括：

- 使用欢迎包容的语言
- 尊重不同的观点和经验
- 友善接受建设性批评
- 关注对社区最有利的事
- 表现同情心和尊重

不可接受的行为包括：

- 使用性暗示的语言或图像
- 嘲笑、侮辱/贬低评论以及人身或政治攻击
- 公开或私下骚扰
- 未经明确许可发布他人隐私信息（"人肉搜索"）
- 其他可合理认定为不适当的行为

## 执行

对不可接受行为的投诉可以通过向 [conduct@example.com](mailto:conduct@example.com) 发送邮件向项目团队报告。所有投诉都将被审查和调查，会产生认为必要和适当的回应。项目团队有义务为事件的记者保持机密。

不遵守或强制实施本行为准则的维护者可能会受到社区领导层决定的临时或永久影响。
