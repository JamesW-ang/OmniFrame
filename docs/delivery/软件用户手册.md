## 产品简介

OmniFrame 是一款面向精密制造行业的工业自动化上位机开发平台，主要用于：

- 运动控制与协调
- PLC 通信与监控
- 视觉检测与数据采集
- 生产管理与报表统计

### 主要特性

| 特性 | 说明 |
|:---|:---|
| 多工站协同 | 支持多个工作站同时运行，状态实时监控 |
| 运动控制 | 支持固高、雷赛、汇川等多种运动控制卡 |
| PLC 通信 | 支持 Modbus TCP/RTU、三菱 MC 等协议 |
| 数据管理 | 实时数据采集、存储与报表导出 |
| 远程监控 | WebSocket + REST API 远程监控接口 |

## 安装指南

### 安装步骤

1. **获取安装包**
   从供应商处获取 `OmniFrame_v3.0.0_Setup.exe` 安装包。

2. **运行安装程序**
   双击安装包，按提示完成安装。建议使用默认安装路径。

3. **配置环境变量**（可选）
   如需连接 MES 系统，请配置以下环境变量：
   ```
   OMNIFRAME_MES_AES_KEY=<32位密钥>
   OMNIFRAME_BARCODE_HOST=<扫码枪IP地址>
   OMNIFRAME_BARCODE_PORT=<扫码枪端口>
   ```

4. **首次启动**
   安装完成后，双击桌面快捷方式启动系统。

## 快速入门

### 登录系统

1. 启动软件后，显示登录界面
2. 输入用户名和密码（默认：`admin` / `admin123`）
3. 点击"登录"按钮进入主界面

### 主界面介绍

主界面分为以下区域：

```
┌─────────────────────────────────────────────┐
│  顶部导航栏  │  Logo  │  用户信息  │ 退出  │
├─────────────────────────────────────────────┤
│                                             │
│              工站状态卡片区                   │
│    ┌────┐  ┌────┐  ┌────┐  ┌────┐        │
│    │工站1│  │工站2│  │工站3│  │工站4│        │
│    └────┘  └────┘  └────┘  └────┘        │
│                                             │
├─────────────────────────────────────────────┤
│  底部状态栏  │  运行状态  │  告警信息  │ 版本  │
└─────────────────────────────────────────────┘
```

### 启动生产

1. 在主界面选择对应工站
2. 点击"启动"按钮
3. 系统进入自动运行模式

## 操作指南

### 启动/停止工站

1. 在工站卡片上点击选中
2. 点击右侧操作按钮
3. 确认操作

### 修改配方

1. 进入"配置管理"
2. 选择"配方"标签
3. 选中要修改的配方
4. 修改参数后保存

### 导出报表

1. 进入"数据管理"
2. 选择查询条件（日期范围、产品类型）
3. 点击"导出"按钮
4. 选择导出格式和保存路径

## 常见问题

### Q1: 登录失败怎么办？

**A**: 检查以下项目：
- 用户名密码是否正确（默认 admin/admin123）
- 是否已创建用户账户
- 联系管理员重置密码

### Q2: 运动控制无响应？

**A**: 按以下步骤排查：
1. 检查运动控制卡是否正常连接
2. 检查限位开关状态
3. 查看日志中的错误信息
4. 尝试复位工站

### Q3: 数据不显示？

**A**: 可能原因：
- 数据库连接异常
- 数据采集服务未启动
- 查询条件范围过小

- 在线工单系统
