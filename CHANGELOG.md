# 版本更新日志

## [1.1.0] - 2024-12-15

### 新增功能
- ✨ **MQTT通信协议支持** - 新增MqttLink组件，支持发布/订阅模式
- ✨ **WebSocket数字孪生增强** - 支持3D模型位置同步和实时告警推送
- ✨ **插件市场** - 官方插件库，一键安装常用功能插件
- ✨ **配置文件加密** - 支持敏感参数加密存储（数据库密码、设备密钥等）

### 改进
- 🔧 **性能优化** - 优化WebSocket推送频率，降低CPU占用率 30%
- 🔧 **稳定性增强** - 改进自动重连机制，支持条件重连策略
- 🔧 **错误处理** - 完善异常捕获和日志记录，便于问题诊断

### 修复
- 🐛 修复了配置管理器在并发写入时的死锁问题
- 🐛 修复了IO管理器读取模拟量时的精度丢失
- 🐛 修复了数字孪生桥接断连后未能正确恢复的问题

### 文档
- 📖 添加MQTT协议集成指南
- 📖 添加插件开发教程

## [1.0.0] - 2024-06-01

### 🎉 首次发布

#### 核心特性
- ✅ **四层架构** - UI/业务逻辑/数据访问/通信层严格分离
- ✅ **反射桥接机制** - 动态加载硬件驱动，升级无需重新编译
- ✅ **WebSocket数字孪生** - 实时推送设备状态到Web前端
- ✅ **单例管理器** - MotionMgr、IoMgr等全局管理器
- ✅ **插件化扩展** - 支持动态加载业务插件
- ✅ **自动重连** - WebSocket断开后自动恢复连接

#### 通信协议支持
- ✅ ModbusTCP/RTU
- ✅ 三菱MC协议
- ✅ 汇川Inovance协议
- ✅ 基恩士HostLink协议
- ✅ OPC UA
- ✅ TCP/UDP通用通信
- ✅ 串口通信

#### 硬件支持
- ✅ 研华运动控制卡（DMC3000/4000）
- ✅ 科运通运动卡（GTS系列）
- ✅ 汇川ECAT从站
- ✅ 主流PLC（三菱、西门子、汇川、基恩士）

#### 组件和工具
- ✅ 配置管理器（XML/INI格式）
- ✅ 日志记录系统
- ✅ 审计日志功能
- ✅ 自定义UI控件库（LED指示灯、数值显示器等）
- ✅ 权限管理系统

## 升级指南

### 从 1.0.x 升级到 1.1.0

1. **备份配置**
   ```bash
   cp -r Config Config.backup
   ```

2. **更新代码**
   ```bash
   git pull origin main
   git checkout v1.1.0
   ```

3. **检查配置更改**
   - 新增MQTT配置项（可选）
   - 新增加密配置项（可选）

4. **重新编译**
   ```bash
   dotnet build --configuration Release
   ```

5. **测试验证**
   - 运行单元测试
   - 在测试环境验证功能
